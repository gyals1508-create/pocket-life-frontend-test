# 일정 페이지 500 Internal Server Error

## 문제 상황
일정 페이지(SchedulePage) 접근 또는 동작 시 서버에서 500 에러(Internal Server Error)가 발생함.

## 발생 위치
- **페이지:** `SchedulePage.jsx`
- **에러 메시지:** `Failed to load resource: the server responded with a status of 500 (Internal Server Error)`

## 상세 정보
브라우저 콘솔에서 `SchedulePage.jsx:1` (또는 관련 API 호출 부)에서 에러가 확인됨.
다음 API 중 하나에서 문제가 발생했을 가능성이 높음:
- `/api/todo/getList`
- `/api/todo/getTodoDates`
- `/api/todo/getHolidays`

## 요청 사항
1. 서버 로그를 확인하여 500 에러의 정확한 원인(NPE, DB 오류 등) 파악 필요.
2. 프론트엔드에서 요청하는 파라미터가 백엔드 기대값과 맞는지 확인 필요 (특히 날짜 포맷 등).
